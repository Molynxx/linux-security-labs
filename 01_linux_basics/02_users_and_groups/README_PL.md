# Users and Groups

## Cel 
Zrozumienie jak Linux zarządza użytkownikami i grupami oraz jakie to ma znaczenie dla bezpieczeństwa.

## Zakres
- `/etc/passwd`
- `/etc/shadow`
- `/etc/groups`
- user management commands

- identyfikacja użytkowników w logach,
- analiza kompromitacji kont.
