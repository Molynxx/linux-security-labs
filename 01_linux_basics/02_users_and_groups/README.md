# Users and Groups

## Cel
Zrozumienie jak Linux zarządza użytkownikami i grupami oraz jakie ma to znaczenie dla bezpieczeństwa systemu. 

## Zakres

- `group`,
- `linux_system_user_basic`,
- `passwd`,
- `shadow`,
- `user_management_commands`.

## Dlaczego to ważne (SOC/IR)
- kontrola dostępu i przydzielanie uprawnień w systemie, 
- identyfikacja użytkowników w logach, 
- analiza incydentów związanych z kompromitacją kont.
